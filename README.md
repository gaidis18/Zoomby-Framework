# Zoomby Framework
An improved version of the ghost framework, in constantly evolving.

# Getting started

## Zoomby installation

> cd zoomby

> chmod +x install.sh

> ./install.sh

## Zoomby uninstallation

> cd zoomby

> chmod +x uninstall.sh

> ./uninstall.sh

***

# Zoomy Framework execution

    To execute Zoomby Framework you 
    should execute the following command.

> python3 zoomby
